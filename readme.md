# Filter componant

Un module pour gérer des filtres


## Pré-requis

NodeJs + npm


## Installation

```$ npm install``` pour installer de quoi compiler les css.


## Commandes usuelles

```$ gulp make:css``` pour compiler les css.

```$ gulp watch``` pour lancer le watch.
